#### 介绍
基于javaweb教务管理系统，前端采用jsp+layui，三个角色：管理员、教师、学生，代码规范整洁，技术选型也非常经典，合适初学java编程者学习以及企业级项目的入门。


#### 软件架构
前端：jsp | jquery | layui  
后端：spring | springmvc | mybatis  
环境：jdk1.8 | maven | mysql
